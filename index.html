Calculator

a novel unit/currency conversion and arithmetical expression evaluator app (that also does Estonian address queries). Input is via Estonian speech, but don't worry, you don't need to know all the 14 cases...

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/Kaljurand/Unitconv

get the source code on GitHub : Kaljurand/Unitconv
